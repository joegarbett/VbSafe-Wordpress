== Description ==

Captures and records admin panel changes. Which options are records are managed on www.vbsafe.co.uk

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/vbsafe` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. From your www.vbsafe.co.uk account, generate an API token (and keep it handy).
4. Use the Settings->VbSafe screen to set the API token for the plugin.

== Frequently Asked Questions ==

= Is it secure? =

Absolutely. We encrypt the data before it is transmitted between systems. 

= How do I see recent changes =

Log into your vbsafe.co.uk account, or visit the GIT repository where your changes will be recorded.
